# Claude Skills Configuration

This directory contains Claude Code skills and configuration for this project.

## Skills

### branding-rules.md
Enforces strict branding and modification policies including:

- **Modification Policy**: Do not add, edit, or remove any element without explicit user request
- **Font Standard**: San Francisco font family only (weights: 100, 200, 400, 500, 600, 700)
- **Color Standard**: Primary color #00c307 (Vibrant Green) only
- **Core Rules**: Demo folder is read-only, all components from demo/, no new colors/fonts
- **Workflow**: Step-by-step process for UI development
- **Conventions**: Branch naming, PR formats, and freezing procedures

## Usage

Claude Code will automatically load these skills when working in this repository. All contributors must follow these branding rules without exception.
